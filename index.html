Boxes is a multiplayer snake-like game developed by Glen Takahashi in his free time. You play and compete with every other on the user on the website at the same time, so you are all trying to race for the same dots. You can see their scores and names on the side of the screen.

It is written in HTML5 with a node.js backend and hosted on Heroku. You can view the source code on GitHub.
